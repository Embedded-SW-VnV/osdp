Functor LinExpr.MakeScalar

module MakeScalar: 
functor (L : S) -> Scalar.S  with type t = L.t

Gives an interface Scalar.S to linear expressions. Since to_float and div have no meaning for linear expressions, they are implemented as assert false. You should never use them. Simialr behavior for the moment for comparison functions. Moreover, the product of two linear expressions is a linear expression only when at least one of them is a constant. LinExpr.Not_linear is raised otherwise.
